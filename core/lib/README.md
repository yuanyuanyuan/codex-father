# 共享库 (`lib`)

该目录是 `core`
模块的共享库，提供了整个应用程序可以复用的通用功能、数据模型和工具。

## 目录结构

- `errors/`: 定义了自定义错误类和错误处理机制。
- `models/`: 包含了应用程序中使用的数据模型和类型定义。
- `queue/`: 实现了任务队列系统，用于异步处理任务。
- `storage/`: 提供了用于持久化存储（如配置和日志）的抽象。
- `tests/`: 包含了 `lib` 模块的单元测试。
- `utils/`: 提供了一系列通用工具函数。
- `validation/`: 包含了用于数据验证的组件。
- `presets.ts`: 定义了预设配置。
- `types.ts`: 定义了 `lib` 模块中使用的通用 TypeScript 类型。
